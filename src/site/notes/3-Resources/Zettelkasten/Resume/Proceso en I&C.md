# Que es un proceso
*Transformación de materia prima*
Conjunto de operaciones necesarias para modificar las características de las materias primas (forma, resistencia, densidad,
tamaño, concentración, estética,etc)

>Puede definirse como un conjunto de operaciones de una planta industrial o como una única operación realizada por una sola máquina,
>herramienta, o etapa de la planta.

- Para la supervisión, monitoreo y control de un proceso industrial se debe obtener información proveniente del mismo.
	- Dicha información se transmite en forma de señales temporales
